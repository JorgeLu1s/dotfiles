# Personal Dotfiles

## Clone this Repo

```git clone git@github.com:JorgeLu1s/dotfiles.git```

## Symlink Dotfiles

```stow <package> -t ~/```

## Neovim LSP servers

To install a Language Server for your language:

```:LspInstall [language-server]```

## Tree-sitter

To install syntax highlighting and tresitter support for your language:

```:TSInstall <TAB>```
